# Required LaTeX Packages for HAFiscal Project
# Last updated: 2025-11-24 (added beamer document class for HAFiscal-Slides.tex)
#
# This file lists all LaTeX packages required to compile HAFiscal.tex and HAFiscal-Slides.tex
# with ONLY texlive-latex-base + texlive-latex-recommended installed.
#
# Packages marked with ✓ were verified essential through iterative compilation.
# Packages marked with ❌ were removed as unnecessary.
#
# Total packages needed beyond base+recommended: ~48 (includes beamer and pgf for presentations)

amsfonts
amsmath
amssymb
amsthm

dsfont
mathtools
