# Contributing to HAFiscal

## ⚠️ STOP! Are you trying to edit `econ-ark/HAFiscal`?

**If YES:** You're in the wrong repository!

`econ-ark/HAFiscal` is **automatically generated** and **read-only**.

**Please go here instead:** https://github.com/llorracc/HAFiscal-Latest

---

## Correct Repository for Contributions

**Source Repository:** `llorracc/HAFiscal-Latest`  
**Link:** https://github.com/llorracc/HAFiscal-Latest

All contributions must be made to `HAFiscal-Latest`, NOT to `econ-ark/HAFiscal`.

---

## Why?

```
llorracc/HAFiscal-Latest (SOURCE - edit here!)
    ↓ automated generation
econ-ark/HAFiscal (GENERATED - read-only)
```

Changes made to `econ-ark/HAFiscal` will be **overwritten and lost** during the next sync.

---

## How to Contribute

### 1. Clone the SOURCE repository:
```bash
git clone https://github.com/llorracc/HAFiscal-Latest.git
cd HAFiscal-Latest
```

### 2. Create a branch:
```bash
git checkout -b feature/my-contribution
```

### 3. Make your changes:
```bash
# Edit files
git add .
git commit -m "Description of changes"
```

### 4. Push and create PR:
```bash
git push origin feature/my-contribution
# Then create PR on GitHub to llorracc/HAFiscal-Latest
```

### 5. After merge:
Changes will flow through the pipeline:
```
HAFiscal-Latest → HAFiscal-Public → econ-ark/HAFiscal
```
